# MPTech Windows Tools

A growing collection of small, portable Windows utilities built for technicians, sysadmins, developers, IT students and advanced users.

Each tool has its own repository, source code and GitHub Release page.

No SaaS. No accounts. No backend servers. No complex setup.

## Available tools

### Audio Device Switcher

Portable Windows utility for quickly switching the default audio output device.

Useful for switching between speakers, headphones, monitors, HDMI audio, USB headsets or virtual devices without opening the Windows sound settings.

- Repository: https://github.com/xml2811/Audio-switcher
- Download latest release: https://github.com/xml2811/Audio-switcher/releases/latest

### Link Downloader

Portable Windows utility for detecting and downloading direct links from pasted text.

Useful for pasting text with many URLs, detecting direct download links and downloading several files locally.

- Repository: https://github.com/xml2811/Link-downloader
- Download latest release: https://github.com/xml2811/Link-downloader/releases/latest

### MPTech Network Tools

Portable Windows toolkit for network diagnostics, local network scan, ping, TCP port tests, traceroute and TXT reports.

- Repository: https://github.com/xml2811/Network-Tools
- Download latest release: https://github.com/xml2811/Network-Tools/releases/latest

### MPTech SSL Checker

Portable Windows utility for checking HTTPS/TLS certificates, SSL health, hostname coverage, SANs, fingerprints and certificate expiration.

- Repository: https://github.com/xml2811/SSL-Checker
- Download latest release: https://github.com/xml2811/SSL-Checker/releases/latest

## Philosophy

Each tool should be:

- Small.
- Useful.
- Portable.
- Fast to use.
- Easy to maintain.
- Independent from servers.
- Focused on one clear problem.

## Notes

Windows SmartScreen may show a warning because the executables are not code-signed yet.

Use each tool responsibly and only in environments where you have permission.
