Audio Device Switcher - MPTech Tools
===================================

Portable Windows tool to quickly switch the default audio output device.

Files:
- audio-device-switcher.exe
- checksums.txt

Current features:
- Detect Windows output audio devices.
- Show current default device.
- Set default output device by button.
- Cycle to next device.
- Configurable global shortcut.
- Exclude devices from shortcut cycle.
- Tray/background mode.
- Close window to keep running in tray.
- Open/close from tray menu.
- Optional launch with Windows.
- Optional start minimized.
- English, Spanish and Portuguese UI.

How to use:
1. Run audio-device-switcher.exe.
2. Choose an output device.
3. Click Predeterminar / Set default.
4. Configure a shortcut if needed.
5. Close the window to keep it running in the tray.
6. Use the tray icon to open or fully close the app.

Notes:
- If the configured shortcut is already used by Windows or another app, choose another shortcut or free it first.
- If you move the portable EXE after enabling startup, disable and enable startup again so the path updates.
- Windows SmartScreen may warn because the EXE is not code-signed yet.
