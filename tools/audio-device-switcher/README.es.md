# Audio Device Switcher

Microherramienta portable para Windows creada por MPTech Tools.

Audio Device Switcher detecta los dispositivos de salida de audio disponibles y permite cambiar rápidamente el dispositivo de salida predeterminado de Windows.

## Estado

Versión inicial en desarrollo.

## Funciones previstas para V1

- Listar dispositivos de salida de audio disponibles.
- Mostrar el dispositivo de salida actual.
- Cambiar el dispositivo predeterminado con un botón.
- Atajo de teclado global para alternar dispositivos.
- Atajo configurable si es viable técnicamente.
- Inicio automático con Windows.
- Inicio minimizado.
- Configuración local mínima.
- Sin login.
- Sin servidor.
- Sin base de datos.
- .exe portable.
