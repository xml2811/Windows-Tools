# Audio Device Switcher

Portable Windows micro tool by MPTech Tools.

Audio Device Switcher detects available audio output devices and lets you quickly switch the default Windows output device.

## Status

Initial development version.

## Planned V1 features

- List available audio output devices.
- Show current default output device.
- Switch default output device with one click.
- Global hotkey to cycle devices.
- Configurable hotkey if technically viable.
- Launch at Windows startup.
- Start minimized.
- Local settings only.
- No login.
- No server.
- No database.
- Portable .exe.
